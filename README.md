## Descripción

Proyecto Android desarrollado con Kotlin y Jetpack Compose para la ruta de aprendizaje **Compila apps fabulosas**.

Incluye los ejercicios:

1. Temas de Material con Jetpack Compose.
2. Animación simple con Jetpack Compose.

## Funcionalidades implementadas

- Lista desplazable de perros con `LazyColumn`.
- Diseño con Material 3.
- Tema personalizado de color y tipografía.
- Tarjetas con imágenes, nombre y edad.
- Animación simple al expandir cada tarjeta.
- Cambio animado de color con `animateColorAsState`.
- Expansión suave con `animateContentSize`.
